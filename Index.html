Tema: Colas y Pilas

Las colas son una estructura de datos que procesan elementos en orden de llegada. Las operaciones principales son encolar, desencolar, ver el primer elemento y comprobar si está vacía, útiles en la gestión ordenada de tareas o procesos.

Las pilas son una estructura de datos que almacenan elementos en orden inverso al de llegada. Las operaciones principales son apilar, desapilar, ver el último elemento y comprobar si está vacía, útiles en la gestión de memoria o en el manejo de funciones.
